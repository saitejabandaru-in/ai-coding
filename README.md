## The AI Coding Stack

- **Planning:** ChatGPT / Claude (architecture, design)
- **Coding:** Cursor / Copilot (implementation)
- **Review:** Claude (code review, security)
- **Testing:** AI-generated tests + human validation
- **Docs:** AI-generated docstrings + README

## Tool Selection Guide

| Task | Best Tool | Why |
|---|---|---|
| Architecture design | Claude / ChatGPT | Better reasoning |
| Inline completion | Copilot / Cursor | In-editor speed |
| Codebase refactoring | Cursor Agent / Claude Code | Multi-file |
| Debugging | Claude / Cursor Chat | Context + reasoning |
| Test generation | Copilot / Cursor | Fast iteration |
| Code review | Claude | Detailed analysis |

## 10 AI Coding Techniques

1. **Comment-First Development** — Write intent, let AI code
2. **Type Hint Guidance** — Strong types guide AI
3. **Test-Driven AI** — Write tests, AI implements
4. **Rubber Duck Debugging** — Explain bug to AI
5. **Incremental Refactoring** — One class at a time
6. **Prompt Chaining** — Design > Skeleton > Implement > Test > Doc
7. **Code Review Mode** — Find bugs before running
8. **Explain and Learn** — Ask AI what it just wrote
9. **Alternative Solutions** — Ask for 3 approaches
10. **Security Scan** — Ask AI to find vulnerabilities
